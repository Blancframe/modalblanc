Modalblanc

Simple Modalbox

View on Github

Getting started

Add script to head or to bottom of body.

Add stylesheet to head

Install via NPM

npm install modalblanc --save

Set content

const modal = new Modalblanc({
    content: "
Simple modal with data!
"
});

Modal 

var modal = new Modalblanc({
    content: "
Content data man!
",
    sideTwo: {
        content: "
Hello universe!
"
    }
});

Modal with second side
